# ミラクル・ラリー！キャンディ・マジック

このプロジェクトは、React + TypeScript + Vite で作成された卓球ゲーム「ミラクル・ラリー！キャンディ・マジック」のリポジトリです。

## 主な機能・特徴
- **ゲームの進行**: 難易度選択画面、ラケット選択画面を通じたゲームプレイ。
- **物理演算**: キャンバス（Canvas）を使用した球の軌跡、スマッシュの速度調整、ネット衝突判定。
- **UI/UX**: スムーズな画面遷移、3Dラケットレンダリング、リッチなアニメーションと背景グラフィック。

## 技術スタック
- **フロントエンド:** React 19, TypeScript
- **ビルドツール:** Vite
- **スタイリング:** CSS (Canvasと組み合わせたリッチなUI表現を含む)

## ローカル環境での動かし方

プロジェクトをローカルで実行するには以下の手順に従ってください。

1. **依存パッケージのインストール**
   ```bash
   npm install
   ```

2. **開発サーバーの起動**
   ```bash
   npm run dev
   ```
   起動後、ブラウザで `http://localhost:5173` (またはコンソールに表示されるURL) にアクセスしてゲームをプレイできます。

3. **本番環境用ビルド**
   ```bash
   npm run build
   ```
   `dist` フォルダに公開用のファイルが出力されます。
